# OPL Website

This folder contains the static OPL website and a small local playground server.

Start the playground server from the repository root:

```powershell
python website/playground_server.py --port 8787
```

Then open:

```text
http://127.0.0.1:8787
```

The server uses the existing OPL Python package to run submitted code locally. It is intended for local development and demos only.
